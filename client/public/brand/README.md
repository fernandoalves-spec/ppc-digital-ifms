# Ativos oficiais da marca IFMS

Adicione nesta pasta os arquivos oficiais conforme os nomes abaixo:

- `ifms-logo-horizontal.svg`
- `ifms-logo-vertical.svg`
- `ifms-icon.svg`

Observacoes:

- Use os arquivos oficiais sem alteracao de proporcao ou cor.
- Em fundos complexos, prefira variacoes com base branca conforme o manual.
- Se os arquivos nao existirem, o frontend usa fallback visual temporario.
