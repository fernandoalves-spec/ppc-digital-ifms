# PPC Digital — IFMS

Projeto Pedagógico de Curso (PPC) Digital do Instituto Federal de Mato Grosso do Sul (IFMS).

## Sobre

Este repositório contém o site do **PPC Digital do IFMS** — uma representação digital do Projeto Pedagógico de Curso para o Curso Superior de Tecnologia em Análise e Desenvolvimento de Sistemas (TADS) oferecido pelo Instituto Federal de Mato Grosso do Sul.

## Estrutura do Projeto

```
ppc-digital-ifms/
├── index.html          # Página principal com todo o conteúdo do PPC
├── css/
│   └── style.css       # Estilos com identidade visual do IFMS
├── js/
│   └── main.js         # Interatividade (busca, navegação, acessibilidade)
└── README.md
```

## Funcionalidades

- 📋 **Sumário lateral** com navegação rápida entre seções
- 🔍 **Busca no conteúdo** com destaque das ocorrências
- 📂 **Seções expansíveis/recolhíveis** para melhor organização
- ♿ **Acessibilidade**: ajuste de fonte, alto contraste
- 🖨️ **Impressão / exportação para PDF**
- 📱 **Responsivo**: funciona em desktop, tablet e celular
- 🏛️ **Identidade visual IFMS** (paleta de cores verde institucional)

## Seções do PPC

1. Identificação do Curso
2. Apresentação
3. Bases Legais
4. Justificativa
5. Objetivos do Curso
6. Perfil do Egresso
7. Organização Curricular (grade curricular por semestres)
8. Metodologia
9. Sistema de Avaliação
10. Estágio Supervisionado
11. Atividades Complementares
12. Trabalho de Conclusão de Curso (TCC)
13. Infraestrutura
14. Corpo Docente
15. Bibliografia de Referência

## Como Usar

Abra o arquivo `index.html` em qualquer navegador web moderno.

## Tecnologias

- HTML5 semântico
- CSS3 (variáveis CSS, Grid, Flexbox, animações)
- JavaScript vanilla (sem dependências externas)
